## вңҰ Overview

**ZAFRAN** is a fully functional, single-page men's fashion e-commerce storefront вҖ” built with **pure HTML, CSS & Vanilla JavaScript**. Zero frameworks. Zero dependencies. One file. Fully deployed.

It delivers a complete shopping experience: browse 6 curated fashion categories, filter across 62+ products, manage a live cart, wishlist favourites, and place orders вҖ” all inside a single `index.html`.
